# 💡 Day 04 Learning

* `ls` is used to view files and folders in the current directory.

* `ifconfig` shows the network details and IP address of my system.

* `apt install net-tools` installs networking commands like `ifconfig`.

* `top` shows the live status of CPU, memory, and running processes.

* `top -h` displays the help page for the `top` command.

* `htop` is an easier and more interactive version of `top`.

* `wget` downloads files directly from the internet using the terminal.

* `uptime` tells how long the system has been running.

* `free` shows the available and used RAM in the system.

* `last` displays the login history of users.

* `apt install wtmpdb` installs the package required to manage login history on some Linux distributions.

* `who` shows who is currently logged into the system.

* `w` shows logged-in users along with what they are doing.

* `locate` quickly finds files without manually searching folders.

* `apt install plocate` installs the `locate` command if it is not already available.

* `cd` changes from one directory to another.

* `ping` checks whether a website or server is reachable.

* `netstat` displays network connections and open ports.

* `ss` is a faster and modern alternative to `netstat`.

* `dig` finds the IP address and DNS information of a website.

* `nslookup` is another command used to look up DNS records.

* `route` shows how the system sends network traffic to other networks.

* `traceroute` shows every network hop between my system and the destination.

* `apt install inetutils-traceroute` installs the `traceroute` command.

* `apt install traceroute` installs the latest traceroute package.

* `hostname` displays the system name.

* `curl` retrieves data from websites or APIs directly from the terminal.

* `mtr` combines `ping` and `traceroute` to monitor network connectivity in real time.

* `whois` provides registration details of a domain name.

* `apt install whois` installs the `whois` command.

* `ifplugstatus` checks whether a network cable is connected.

* `apt install ifplugd` installs the package required for `ifplugstatus`.

* `vi` is used to create and edit files directly in the terminal.

* `cat` displays the contents of a file.

* `mv` renames a file or moves it to another location.

* `./linux.sh` executes the shell script from the current directory.

* `ls -l` shows detailed information like permissions, owner, and file size.

* `chmod` changes file permissions such as read, write, and execute.

* `history` displays all previously executed commands in the terminal.
