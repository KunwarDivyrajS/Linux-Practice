# 💡 Day 05 Learning

* `dmesg` shows system messages generated during boot and hardware detection.

* `dmesg | wc -l` counts the total number of system log messages.

* `ls` lists files and folders in the current directory.

* `lsblk` shows all available disks and their partitions.

* `ls -l` displays detailed information like permissions, owner, and file size.

* `cat /proc/cpuinfo` provides detailed information about the CPU.

* `free -h` shows RAM usage in a human-readable format.

* `df -h` displays available and used disk space for each mounted drive.

* `lshw` provides detailed information about the system hardware.

* `fdisk -l` lists all disks and their partition details.

* `du` shows how much storage a file or folder is using.

* `du -ah` displays the size of all files and folders.

* `du -sh` shows only the total size of a folder in a simple format.

* `history` displays previously executed terminal commands for quick review.
